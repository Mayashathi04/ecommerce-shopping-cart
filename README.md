# 🛒 Maya's Cart - Full-Stack E-Commerce Application

A responsive full-stack e-commerce marketplace featuring a clean user interface, dynamic cart calculations, an expanded product catalog, and automated customer profile validation routines.

## ✨ Key Features
* **Expanded Electronic Catalog:** Displaying dynamic product structures formatted perfectly in Indian Rupees (₹) using native regional numeric formatting.
* **Smart Delivery Settings Engine:** Automated location tracking that triggers custom geo-data lookups and overrides based on the user's entered 6-digit Pincode.
* **Inline Validation Notice System:** Replaced standard browser popup windows (`alert()`) with native UI elements, providing seamless inline green success notifications and red validation error frames for a modern user experience.
* **Persistent State Management:** Synchronizes active customer profiles and order histories securely across sessions using browser storage systems.

## 🛠️ Tech Stack
* **Frontend:** React, HTML5, CSS3, JavaScript (ES6)
* **Backend Configurations:** Node.js, Express
* **Styling & Layout:** Component-driven CSS architecture optimized for responsive grids and card layouts.
